# 关于我

这里是 **Perseverance** 的个人安全博客，主要记录：

- CTF 比赛复盘
- Web 安全题解
- 数字取证与杂项题分析
- 解题脚本、踩坑记录和思路沉淀

## 为什么做这个站

很多比赛里的解法如果不及时整理，很快就会只剩下一个模糊印象。这个站点的目标，就是把一次次解题过程转成可以复查、可以复用、也方便继续扩展的知识资产。

## 当前内容方向

- 比赛 Writeup
- 关键利用链拆解
- 还原过程中的截图与脚本
- 后续可能补充专题化总结

## 站点说明

本站使用 [VitePress](https://vitepress.dev/) 构建，主题参考 [Theme Sugarat](https://theme.sugarat.top/)，并针对 GitHub Pages 项目站点做了路径和样式适配。
