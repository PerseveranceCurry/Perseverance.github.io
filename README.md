# Perseverance.github.io

基于 [VitePress](https://vitepress.dev/) 和 [@sugarat/theme](https://theme.sugarat.top/) 搭建的个人 CTF / 安全博客。

## 本地开发

```bash
npm install
npm run import:writeups
npm run dev
```

## 构建

```bash
npm run build
```

站点默认部署到 GitHub Pages 项目站点：

`https://perseverancecurry.github.io/Perseverance.github.io/`
